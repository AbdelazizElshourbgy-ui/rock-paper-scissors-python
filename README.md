# 🎮 Rock Paper Scissors Game

A simple and interactive Rock Paper Scissors game built using Python.  
The player competes against the computer in a classic game of chance.

---

## 📌 About The Project

This project is a beginner-friendly Python application that allows users to play Rock, Paper, Scissors against the computer.  
It demonstrates the use of:

- Conditional statements
- Loops
- Functions
- Random module
- Input validation

The game ensures valid user input and randomly generates the computer’s choice.

---

## ✨ Features

- 👤 User vs Computer gameplay  
- 🎯 Input validation to prevent errors  
- 🔁 Uses Python's random module  
- 🧠 Beginner-friendly project  
- ⚡ Simple and clean logic  

---

## 🛠️ Technologies Used

- Python
- Loops & Conditions
- Functions
- Random Module
